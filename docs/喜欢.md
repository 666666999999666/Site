# Website UI Redesign Specification

## 目标

请将当前网站整体 UI 重构为一个：
极简、高级、现代的个人开发者博客网站

设计参考：
- Vercel 官网设计语言
- Linear App 风格
- shadcn/ui 组件风格（base-nova / neutral 色系）

目标关键词：minimal / clean / professional / developer-focused / content-first / high whitespace / black-white-gray

不要设计成：炫酷科技风 / 渐变背景 / 大量动画 / 卡片堆叠 / 花哨颜色

---

## 1. 全局设计系统

### Color System（OKLCH 色彩空间）

所有颜色通过 CSS 变量定义，使用 OKLCH 色彩空间（chroma=0 纯中性色），通过 shadcn 的 @theme inline 映射到 Tailwind 类名。

亮色模式：
- Background: #FFFFFF (oklch(1 0 0))
- Primary Text: #252525 (oklch(0.145 0 0))
- Secondary Text: #8A8A8A (oklch(0.556 0 0)) — muted-foreground
- Border: #EAEAEA (oklch(0.922 0 0))
- Hover Background: #F5F5F5 (oklch(0.97 0 0)) — muted
- Primary Button BG: #252525 (同 Primary Text)
- Destructive: oklch(0.577 0.245 27.325) — 唯一带色相的颜色，仅用于危险操作

暗色模式：
- Background: #252525 (oklch(0.145 0 0))
- Primary Text: #FAFAFA (oklch(0.985 0 0))
- Secondary Text: #8A8A8A (oklch(0.708 0 0))
- Border: rgba(255,255,255,0.1) (oklch(1 0 0 / 10%))
- Hover Background: #454545 (oklch(0.269 0 0))
- Primary Button BG: #EAEAEA
- Destructive: oklch(0.704 0.191 22.216)

原则：
- 不使用彩色背景，不使用渐变
- 极少使用阴影（仅灯箱图片 shadow-2xl）
- 使用细边框区分层级，大量使用 50% 透明度边框（border-border/50）营造轻盈感
- 悬停背景使用 10%~50% 透明度的 muted 色，微妙变化
- 状态徽章使用 10% 透明度彩色底 + 对应色文字

 Typography

字体族：
- 正文+标题：Geist Sans（通过 Google Fonts 加载，CSS 变量 --font-sans）
- 代码：Geist Mono（CSS 变量 --font-mono）
- 标题字体同正文，不使用额外装饰字体

字号梯度：
- Hero 标题：48px → 60px (text-5xl sm:text-6xl)
- 页面主标题：30px → 36px (text-3xl sm:text-4xl)
- 区块标题：24px (text-2xl)
- 卡片标题：18px (text-lg) / 20px (text-xl)
- 组件文字/按钮：14px (text-sm)
- 辅助文字/标签：12px (text-xs)
- Markdown 正文：16px (1rem)

字重：
- 标题：700 (bold) / 600 (semibold)
- 按钮/导航/标签：500 (medium)
- 正文：400 (normal)

字间距：
- 所有标题：tracking-tight
- 分类标签/元数据标签：uppercase + tracking-wider

行高：
- Markdown 正文：1.75
- Markdown h1：1.3 / h2：1.35 / h3：1.4
- 卡片标题：leading-snug
- 描述文字：leading-relaxed

HTML 根元素：font-sans antialiased（字体平滑渲染）

 Border & Radius

圆角系统（基于 --radius: 0.625rem）：
- 按钮/卡片/输入框/代码块：rounded-lg = 10px
- 导航链接：rounded-md = 8px
- 标签/徽章/状态指示：rounded-full = 9999px（全圆角）

边框规范：
- 默认边框：1px border-border/50（50% 透明度，轻盈感）
- 悬停边框：1px border-border（100% 不透明，变实）
- 表单输入框：1px border-border（聚焦时 border-ring）
- 代码块标题栏：1px border-zinc-800
- 空状态：1px border-dashed border-border
- 全局：* { border-color: var(--border); outline-color: oklch(0.708 0 0 / 50%); }

 Spacing

容器宽度：
- 主内容区：max-w-5xl = 1024px
- 文章内容：max-w-3xl = 768px
- 登录表单：max-w-sm = 384px
- TOC 侧栏：w-56 = 224px
- Admin 侧栏：w-60 = 240px

页面垂直间距：
- Hero：py-24 sm:py-36（内容自适应，不设固定高度）
- 列表页：py-16
- 详情页：py-12
- Admin：py-8

水平内边距：px-4 sm:px-6（16px → 24px）

网格间距：gap-6 (24px) / gap-8 (32px)

## 11. 交互规范

过渡：所有交互统一 transition-colors 或 transition-all，150ms ease

按钮交互：
- Hover：根据变体改变背景色
- Active（按下）：非弹窗按钮下移 1px (translate-y-px)，松开回弹，物理按压感
- Focus：focus-visible:border-ring + ring-3 ring-ring/50
- Disabled：opacity-50 + pointer-events-none

卡片交互：
- Hover 3 层反馈：边框从 50% 透明变实 + 浅背景浮现 + 标题变主色
- 标题使用 group-hover 实现联动变色

导航交互：
- 链接 hover：bg-accent 淡入，transition-colors 150ms
- 主题切换：disableTransitionOnChange（切换时禁用过渡，避免闪烁）

输入框交互：
- Focus：ring 扩展 + 边框变色，清晰焦点指示
- 搜索框：实时过滤 + 右侧清除按钮

图片交互：
- cursor-zoom-in → 点击打开灯箱
- 灯箱：ESC / 点击遮罩关闭 + 滚动锁定

代码块交互：
- 复制按钮：点击后图标变 ✓ + 文字变化

删除交互：
- 需二次确认，3 秒超时自动恢复默认态

自动保存：
- 5 秒防抖自动保存
- 状态指示器实时显示

## 15. 文案风格

所有文案采用个人开发者技术博客风格：
- 标题用陈述句，不用疑问句
- 摘要不超过两行（line-clamp-2）
- 日期格式：YYYY年M月D日
- 阅读时间：X分钟阅读

不要：欢迎来到我的网站 / 这里展示我的作品
改成：记录思考、项目和对现代 Web 开发的探索。

内容方向：Web 开发 / AI Agent / 开源项目 / 工程实践 / 技术研究

## 最终验收标准

完成后网站应该像：一个 3-5 年经验的独立开发者的技术主页
而不是：模板站 / 企业官网 / SaaS Landing Page / 炫技 Demo

核心感觉：安静 / 专业 / 克制 / 技术感 / 内容优先
